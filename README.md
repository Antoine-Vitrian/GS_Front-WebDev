# Futuro do Trabalho — Global Solution (Front-End)

## Resumo do projeto
Aplicação SPA em React + Tailwind que simula uma rede profissional voltada ao futuro do trabalho. Permite busca, filtros, visualização de perfis, recomendação e envio de mensagens (simulado).
